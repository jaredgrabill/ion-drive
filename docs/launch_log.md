# Launch log — external users, not stars

who actually ran `init`** (or deployed the image). Stars, upvotes, and comments
don't count as users. Lane 4 closes at 5–10 rows with evidence; the go/no-go
note at the bottom is the input to whatever gets unfrozen next.
